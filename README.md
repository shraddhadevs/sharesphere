# ShareSphere: Donation & Reuse Platform

ShareSphere connects donors with verified NGOs to repurpose surplus items like clothes and books. Our goal is to promote sustainability and reduce waste.

## Key Features
- **Secure User Authentication:** Safe login for Donors and NGOs.
- **Donation Dashboard:** Track your donation history and status easily.
- **Verified NGO Directory:** Find trusted organizations to support.
- **Responsive Design:** Optimized for mobile and desktop (Bootstrap 5).

## Tech Stack
- **Frontend:** HTML, CSS, Bootstrap 5
- **Backend:** PHP
- **Database:** MySQL

## Getting Started
1. Clone this repository.
2. Import the `database.sql` file into your local phpMyAdmin.
3. Update your database connection in `config.php`.
4. Run on your local server (XAMPP/WAMP).
